Reflection

Responsive Design in terms of web technology is a design theorem meant to emphasize a

web page's ability to be compatible on different platforms. This means that the web page

would render effectively on various devices with different size screens. To achieve

responsive design using scalable units such as percent and EM’s over fixed units such as

points or pixels is crutial. During my research I read about how to use the scalable

units correctly as i've never had any experience using this kind of design

scheme before. With a growing user base using exclusively mobile devices it is important

that these users have access to websites that are compatible.


I used the one column design for my webpage because it made the most sense for

what I envisioned for my online store. I used percentage scale for the images displayed on

my store front page replacing the pixel units I used previously. I did this by trial and error to

find the correct image dimensions using percent and then with some simple math calculated that

1% = 13.513 px. For the display of my webpage I put the entire thing into

a Div and set it to display at 90% in the css file. To ensure the main content does not

render incorrectly I added another tag "Section" called main that covers the body of the html to display

at 100%. I also did all the css measurements using EM’s instead of pixels to make all the

formatting compatible with any sized display screen.


During this assignment I learned how to incorporate scalable units into the design of my

website and how to follow different design patterns such as the single column design I used

among others.
